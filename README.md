A sleek, responsive personal portfolio showcasing my journey as a B.Tech CSE student and developer.
This project highlights my expertise in Full-stack development, AI/ML integrations, and practical problem-solving.

### ✨ Key Features
- **Modern UI**: Clean design built with vanilla HTML, CSS, and JavaScript.
- **Project Gallery**: Showcasing major works like AI Attendance, IPL Predictor, and Snackscart.
- **Tech Stack**: Highlights proficiency in Python, FastAPI, React, OpenCV, and MySQL.
- **Responsive Design**: Optimized for all devices with smooth navigation and micro-animations.
- **Direct Contact**: Integrated links for Email, LinkedIn, and GitHub for easy networking.
